## 개요
`cursor` 속성을 이용하면 해당 태그 위에 위치하는 마우스 커서의 모양을 바꿀 수 있습니다.

- `auto`: 자동
- `default`: 기본값 (화살표)
- `pointer`: 손가락 모양 (클릭 가능한 버튼)
- `wait`: 로딩

등 다양한 종류의 값이 있습니다.

## 사용법
```css
.btn-wait { cursor: wait }
```
